# Supported Libraries

* HTTP Server
  * `gin`: [Gin](https://github.com/gin-gonic/gin)
  * `http`: [Native HTTP](https://pkg.go.dev/net/http)
  * `go-restfulv3`: [Go-Restful](https://github.com/emicklei/go-restful)
* HTTP Client
  * `http`: [Native HTTP](https://pkg.go.dev/net/http)
* RPC Frameworks
  * `dubbo`: [Dubbo](https://github.com/apache/dubbo-go)
